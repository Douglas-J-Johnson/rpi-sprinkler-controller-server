# rpi-sprinkler-controller-server
This server is adapted from the sample code provide for the Waveshare 8 relay Raspberry Pi Board.\
[Product Page](https://www.waveshare.com/rpi-relay-board-b.htm)\
[Documentation](https://www.waveshare.com/wiki/RPi_Relay_Board_(B))

The included python file when executed will run a python-bottle web server that will allow you to control the relay board.  A simple client application
